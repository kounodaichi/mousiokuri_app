# クリニック申し送りアプリ
<img width="1245" alt="Screen Shot 2023-03-15 at 14 02 09" src="https://user-images.githubusercontent.com/86922727/225539286-a86b074f-34a8-4a6d-9950-78deb782fb08.png">
クリニックや病院での利用者、業務の申し送りをサポートするアプリです。  

## URL  
https://peaceful-shore-53282.herokuapp.com  
ログインページからログイン可能です。  

## 作成背景  
前職では整形外科クリニックにて作業療法士として勤務していました。
医療従事者として働く中で、現場内での「申し送り・情報共有」は患者の予後に多大な影響を与えかねません。
部署が多い医療現場の中で、私が勤務していたクリニックは情報が紙媒体での共有状態でした。その中で情報共有不足によるミスが起こるたびに、何度も情報共有の在り方を考えさせられました。このアプリではそんな申し送り・情報共有不足による医療ミスを防ぐために、デバイスでの申し送りアプリを考案しました。

## ユーザーへの思い
申し送りが一括されていて見やすく、情報不足による医療ミスを無くすこと。まずはその役に立ちたいと思いました。  
そして派生した仕事などの業務が今までよりも円滑に進めて行けたら尚嬉しい限りです。

## こだわった点
サイドバーを追加する等、他部署からの申し送りを管理する上でシンプルに見やすくすることを一番に考えました。また、部署分け投稿やキーワード検索等、便利なオプションを追加しました。アプリを開発するにあたって前職の医療従事者、３名の方に使用していただきフィードバックをいただきました。自分では気づかなかったユーザー視点のアドバイスを聞くことができました。

## アプリ紹介
* 各部署でログイン
* 非ログインでも利用可能にした。(感想投稿,コメント時のみログインページへ）
* 主なメニューはサイドバーに表示
* fontawesomeのアイコンを使用しアプリの機能を分かりやすく表示

### ①トップページ
* アプリの作成経緯、機能の説明
* 前職の先輩方（医療従事者）から当アプリ使用後の実際のフィードバック

## ②投稿一覧
* 申し送り内容のキーワード検索を配置
* 申し送り追加ボタン配置（サイドバーのメニューからも可能）
* 投稿時間、投稿者、宛先、タイトル、閲覧を表にして見やすく表示
* 詳細、編集、削除ボタン配置（編集、削除は投稿者のみ可能）
* 投稿者のアドレスは、ログインアカウントから自動的に反映

## ③申し送り入力フォーム
* 宛先の部署名をセレクトボックスから選択可能
* 画像投稿が可能
* タイトル、内容のいずれかが未入力の場合、エラーメッセージが表示

## ④投稿詳細
* タイトル、内容、添付画像の表示
* 申し送りに対しメッセージ可能

## ⑤部署分け投稿
* 各部署ごとに申しお送りが閲覧可能

## ⑥タスク管理
* タスク管理にて、申し送りから派生した業務を管理可能
* やることリスト、完了済みのタスク一覧を表示
* 投稿者のみ、タスク完了ボタンを押すことが可能
* やることリスト追加時に期限の設定が可能、その後もタスク一覧に表示される
